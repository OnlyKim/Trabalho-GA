COMANDOS:

 - MOVIMENTAÇÃO DA TELA: A, W, S, D
 - ROTAÇÃO DA TELA: Mouse
 - MOVIMENTAÇÃO DO CURSOR: Setas(cima, baixo, esquerda, direita)
 - MOVENDO O CURSOR EM PROFUNDIDADE: O ou L

 - ADICIONAR VOXEL: Espaço
 - REMOVER VOXEL: E
